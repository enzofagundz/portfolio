## ☕ Usando o Portfólio

Para usar o Portfólio, siga estas etapas:

Clone o repositório
```
git clone git@github.com:enzofagundz/portfolio.git
cd portfolio
```

Instale as dependências
```sh
npm install
```

Compile e coloque para desenvolvimento
```sh
npm run dev
```
